# Pull Request Practice Repository

Welcome to this practice repository! 🎉 This space is designed to help you learn how pull requests work using multiple GitHub accounts.

## About This Project
This is a simple practice project where you can safely experiment with:
- Creating branches
- Making changes to files
- Opening pull requests
- Reviewing code
- Merging changes

## Practice Tasks for Beginners

Try these exercises to master pull requests:

### ✅ Task 1: Add Yourself to Contributors
1. Create a new branch
2. Add your second GitHub username under "Current Contributors"
3. Open a pull request

### ✅ Task 2: Create a New File
1. Create a file named `my-first-pr.txt`
2. Write something about why you're learning GitHub
3. Submit as a pull request
